Series Manager.

This program is designed to maintain a thrill and detailed relationship between you and your TV Series, Movies, Animes, and provides you with a list of watched episodes so you won't ever need to open one by one again.

Install the application.

- From Source Code.

First you create a directory where you want to store your copy of series-manager, clone the git repository into your system as shown below.

$ cd directory_you_created/
$ git clone https://github.com/Ramalus/series-manager.git

After the cloning process you can compile and run from source.

- Executing ".jar" executable file.

Download the "Series_Manager.jar" file that can be found in the git repository "https://github.com/Ramalus/series-manager", or click in the "View on GitHub" button on the top right corner of this website, run it, and you are ready to go.
